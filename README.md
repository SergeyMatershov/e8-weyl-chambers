# E₆, E₇, E₈ — Algorithmic Construction and Atlas of Weyl Chambers

## 📖 About the Project
Algorithmic method for constructing exceptional Lie algebras E₆, E₇, E₈ without using Cartan matrices or Dynkin diagrams. All 240 roots of E₈ are generated explicitly. An alternative Weyl chamber is discovered, with mapping to Bourbaki numbering.

## 🔬 Key Results
• **E₈:** 240 roots, 8 simple roots found algorithmically, Bourbaki mapping: [0, 1, 2, 3, 4, 6, 7, 5]
• **E₇** and **E₆** constructed as subsystems
• **E₆:** complete match 6/6 with Bourbaki dimensions
• **0 prime values** among fundamental representation dimensions (k=1..200)
